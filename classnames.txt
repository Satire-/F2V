CLASS NAMES:
============

Unsuppressed rifles:

"UKF_L115A3"
"UKF_L115A3_wdl"
"UKF_L115A3_des"

Suppressed rifles:

"UKF_L115A3_sd"
"UKF_L115A3_sd_wdl"
"UKF_L115A3_sd_des"

Adding the sufix '_fgrip' to these classnames will spawn these rifles with the bipod in the
deployed position.

Magazines:

"UKF_5Rnd_338_Mag"

All rifles use this magazine. There is no sub-sonic ammunition for use with the suppressed
rifles; the suppressor's purpose is to eliminate muzzle-flash.
